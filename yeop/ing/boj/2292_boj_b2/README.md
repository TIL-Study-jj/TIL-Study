# 문제 사이트
BAEKJOON ONLINE JUDGE

# 문제 요약
1. 문제 난이도 : BRONZE 2
2. 알고리즘 분류 : 수학
3. 문제 내용 : 주어진 방으로 가기 위한 최단 경로 찾기

# 구현 로직
1. 중앙에서 출발하면, 중앙을 기준으로 하나의 둘레마다 한칸씩 이동한다.
2. 따라서 특정 방으로 가기 위한 최단 경로는, 몇 번째 둘레인지 확인하면 알 수 있다.
3. 각 둘레마다 가장 작은 값, 가장 큰 값에 규칙이 있으므로 이를 증가시키며 몇 번째 둘레인지 확인한다.
